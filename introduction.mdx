## Overview

George provides a centralized hub for your project documentation. You create, organize, and collaborate on docs without leaving your workflow. Built for teams, it supports version control, search, and integrations to streamline knowledge sharing.

<Callout kind="info">
  George integrates seamlessly with your existing tools like GitHub, Notion, and Slack.
</Callout>

## Quick Start

Get up and running in minutes. Follow these steps to create your first documentation page.

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Visit `https://george.example.com` and create your account with email or GitHub.
  </Step>
  <Step title="Create Workspace" icon="home">
    Set up a new workspace for your project. Name it and add team members.
  </Step>
  <Step title="Add First Doc" icon="plus">
    Click the `+ New Page` button. Use the MDX editor to write your content.
  </Step>
  <Step title="Publish" icon="globe">
    Preview changes, then publish. Share the public link with your team.
  </Step>
</Steps>

<CodeGroup tabs="JavaScript,Python">
```javascript
// Embed George docs in your app
import { GeorgeEmbed } from '@george/docs-sdk';

<GeorgeEmbed workspaceId="your-workspace-id" pageId="doc-123" />
```

```python
# Python integration example
from george_docs import embed_doc

embed_doc(workspace_id="your-workspace-id", page_id="doc-123")
```
</CodeGroup>

## Navigation Guide

Navigate George efficiently using these common paths.

<Tabs>
  <Tab title="Sidebar" icon="menu">
